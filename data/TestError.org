** Text content does not match server-rendered HTML                   :error:
:PROPERTIES:
:CODE: hydration-error
:TAGS: react nextjs hydration
:END:
#+begin_src error
Uncaught Error: Text content does not match server-rendered HTML.
#+end_src
*** Cause                                                             :cause:
This error occurs when the HTML generated by Next.js on the server (SSR) differs from what React attempts to render on the client during the initial hydration pass.
*** Solution                                                       :solution:
Ensure that the initial render is deterministic. If you need to render dynamic data that differs between server and client (like a local timestamp), wait until the component has mounted on the client.
